### **1.1 JavaScript nedir?**

JavaScript (JS), web sayfalarına etkileşim katmak için kullanılan programlama dilidir.
HTML yapıyı verir, CSS tasarımı yapar, JavaScript ise sayfaya hareket kazandırır.

**Temel Özellikler:**

- Tarayıcıda çalışır (Chrome, Firefox, vs.)
- HTML ve CSS ile birlikte kullanılır.
- Dinamik içerik, animasyon, form kontrolü gibi işler yapar.
- Hem frontend (tarayıcı), hem backend (Node.js ile) çalışabilir.

**Kullanım Alanları:**

- Web sitelerinde **görsel animasyonlar** (kayma, renk geçişi gibi efektlerle sayfaya hareket katmak)
- Butona tıklanınca işlem yapmak (örneğin: tıklanınca modal açmak)
- Form verisi kontrol etmek (örneğin: e-posta geçerli mi, şifre uzunluğu yeterli mi, boş alan kaldı mı?)
- API'den veri çekmek ve ekranda göstermek (örneğin: hava durumu bilgisi getirmek)
